# About CupCart

CupCart is an open-source, free (as in speech), e-commerce platform inspired by WordPress.

# Project Goals

The main goal of the project is to be flexible and easily modable. We are taking the best parts of WordPress's
plugin and theme APIs and improving them for use in e-commerce.

# Kohana Source Code

Some of our source code is ported from the KohanaPHP framework project. A copy of the Kohana license is included in the license.md file.

Code ported from Kohana includes:

* Database/DB classes.
